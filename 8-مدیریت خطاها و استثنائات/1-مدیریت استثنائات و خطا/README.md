* استفاده از صفحات خطای شخصی‌سازی شده به جای خطاهای پیشفرض
* عدم نمایش خطاهای Stack و قطعات کد به کاربر
* عدم نمایش خطاهای فنی و فایل لاگ به‌کاربر
* غیرفعال‌سازی بنرهای مربوط به تکنولوژی‌ها
